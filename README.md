# Y4-JDK-8

y4-jdk is an openjdk8u branch that is designed with a great emphasis on security features

All downloadable Release versions are built using Github Actions and have passed jtreg regression tests

Supported versions:
- Windows x64 (JDK/JRE)
- Linux x64 (JDK/JRE)
- MacOS x64 (JDK/JRE)
